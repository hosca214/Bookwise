How it works: The type toggle (All / Income / Expenses) is always first. Selecting Income or Expenses reveals that type's category chips in a scrollable row — each chip toggles independently. Selecting multiple chips ANDs them together (shows rows matching any selected category). Month chips are always visible. Totals update in real time.
Red dot on camera icon = expense row missing a receipt. Green camera = receipt attached. Dot disappears when photo is uploaded.
